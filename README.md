# WatCat?

이 프로젝트는 **Spring Boot**를 사용하여 만든 **포트폴리오** 웹 애플리케이션입니다.

## 기술 스택

- **Backend**: Spring Boot
- **Frontend**: Thymeleaf
- **Database**: MySQL
- **Infra**: AWS
